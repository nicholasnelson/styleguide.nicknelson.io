Living Style Guides

What is a Living Style Guide?

A living style guide is a tool used to document the style and patterns used in user interface projects. This may include things such as:

  • Color pallett choices
  • Font choices
  • Accessibility guidelines
  • Layout guidelines

The living style guide dictates how pages should look and feel.

Why is a Living Style Guide useful to developers and designers?

A living style guide is useful to both developers and designers as it helps to keep these two groups 'on the same page', leading to reduced chance of confusion if, for example, one group is following one convention contrary to the other.

It also improves usability through a uniform style/theme used throughout the application/site.

Living Style Guide Tips

  • Keep guidelines simple and easy to understand.
  • Avoid possible conflicts/overlaps between style requirements.
  • Keep it DRY (Don't Repeat Yourself), define requirements only once for any element.
  • Strict adherence, 'exceptions' to the guide should not exist and any need for this should trigger reanalysis/reworking of the guide.
  • Keep it updated. It is a living style guide and should adapt to fit requirements.
